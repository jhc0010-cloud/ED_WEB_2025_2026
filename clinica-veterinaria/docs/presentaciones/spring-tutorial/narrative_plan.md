# Tutorial Spring Boot - Clinica veterinaria

## Audiencia

Equipo de desarrollo del proyecto de clinica veterinaria. La presentacion debe servir para explicar en clase o al equipo que se va a construir con Spring Boot y que funcion cumple cada parte del backend.

## Objetivo

Crear una presentacion editable en PowerPoint que explique:

- Que vamos a hacer con Spring Boot.
- Que capas y archivos se van a usar.
- Como viaja una peticion desde el frontend hasta MySQL.
- Que funciones tendran controladores, servicios, repositorios, entidades, DTO, seguridad y roles.
- Cual sera el plan de trabajo recomendado.

## Narrativa

La historia empieza con el objetivo funcional del proyecto, baja a la arquitectura por capas, explica cada pieza con ejemplos reales del repositorio y termina con una ruta de trabajo practica para implementar nuevas funcionalidades sin perder orden.

## Diapositivas

1. Portada del tutorial.
2. Que vamos a construir.
3. Mapa general de Spring Boot.
4. Flujo de una peticion.
5. Maven y dependencias.
6. Configuracion principal.
7. Entidades y base de datos.
8. Repositorios.
9. Servicios.
10. Controladores y API REST.
11. Seguridad, login y roles.
12. Frontend, plan de trabajo y buenas practicas.

## Sistema visual

Estilo claro, docente y tecnico. Paleta con fondo claro, tinta oscura, verde Spring, azul petroleo, coral y amarillo suave. Tipografias modernas sin serif: Poppins para titulos y Lato para cuerpo.

## Editabilidad

Todo el texto importante, tablas, tarjetas, flechas, diagramas y codigo se crean como objetos editables de PowerPoint. Las notas de ponente incluyen explicaciones para usar la presentacion como tutorial.
